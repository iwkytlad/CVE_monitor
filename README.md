**CVE Monitor** ─ инструмент для сканирования установленного на хосте ПО на наличие известных уязвимостей (CVE) с генерацией HTML-отчётов, JSON-результатов и текстовых алёртов для критических уязвимостей.

---

## Проект автоматически:

- Сканирует установленное ПО (Windows, Linux, macOS)
- Сравнивает версии ПО с базой известных CVE
- Генерирует HTML-отчёт с таблицей уязвимостей
- Создаёт JSON-файл с результатами сканирования
- Формирует текстовые алёрты для критических уязвимостей (severity: critical)

---

## Структура проекта

<details>
<summary>Нажмите, чтобы развернуть</summary>

```
cve_monitor/
├── data/
│   ├── hot_cves.json              # База CVE (пока нет автообновления)

└── README.md                      # Документация
```

</details>

## Требования

- Python 3.8+
- Установленные менеджеры пакетов (для автоматического сканирования):
  - **Windows:** `winget`
  - **Linux:** `dpkg`
  - **macOS:** `brew`
- Подготовить CVE базу (пока нет автозагрузки)

---
